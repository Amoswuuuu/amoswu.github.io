Chapter7.硬盘和内存

硬盘的命名方式

  1. 一般主板上有两个IDE接口，一共可以安装四个IDE设备。Linux下对IDE设备的命名是以 hd开头，分别是hda、hdb、hdc、hdd。
  2. SCSI接口设备是用 sd命名，第一个是sda，第二个是sdb，以此类推。

硬盘的分区

  1. 查看分区表 sudo fdisk -l
  2. 创建分区 sudo fdisk /dev/sdb ,根据提示操作
  3. 格式化 sudo mkfs -t [文件系统格式] 设备名
  4. 卸载 sudo umount 设备名或挂载点

硬盘配额

硬盘配额就是管理员可以为用户所能使用的磁盘空间进行配额限制，每一用户只能使用最大配额范围内的磁盘空间，在Linux系统发行版本中使用quota来对用户进行磁盘配额管理，避免了某些用户因为存储垃圾文件浪费磁盘空间导致其他用户无法正常工作。

软限制：在不超出硬限制的情况下，用户在规定时间内释放空间可继续使用。
硬限制：一个用户可拥有的磁盘空间或文件的绝对数量，绝不允许超过这个限制。

  1. 查看内核是否支持配额 # grep CONFIG_QUOTA /boot/config-[版本号]
  2. 安装磁盘配额工具 sudo apt-get install quota quotatool
  3. 激活分区的配额功能
    1). 建立目录 # mkdir /myquota
    2). 修改属性 # chown username: username /myquota
    3). 修改/etc/fstab 文件

任务计划

  • 执行一次的 at 命令
    • 格式 at [选项] [时间日期]
  • 任意时间执行的 batch 命令
    • 格式 输入batch命令后，机器显示 at> ，输入机器执行的内容，在系统不忙时会自动执行。
  • 在指定时间执行的 crontab 命令
    • 格式 crontab [选项] [用户名]
